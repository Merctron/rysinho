# rysinho

WIP experiments on LLM brain surgery for small language models. Inspired by [this work](https://dnhkng.github.io/posts/rys/#the-brain-scanner) for `rys` (Repeat Your Self). I added the ['inho'](https://en.wiktionary.org/wiki/-inho) since this is aimed at small language models.

Currently this just houses some code and docs generated based on some back-and-forth with Opus-4.6. The goal is to modify this with a better understanding of model architecture and hand write a large part of the implementation to learn the fundamentals of transformer architecture and take this in novel directions.
